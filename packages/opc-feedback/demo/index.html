opc-feedback Component

opc-feedback component enables to collect the feedback from the user.

Supported Params

  1. url - URL of your hosted feedback spa. By default /feedback is the url.

Themes

  1. opc-feedback in red(default) color
    2. HTML 
     <opc-feedback></opc-feedback>
  2. opc-feedback in blue color
    3. HTML 
    <div class="blue">
     <opc-feedback></opc-feedback>
</div>

Event Handling

  1. Obtaining the data from opc-feedback Component after the component action is initiated
    2. Javascript 
    document.querySelector('opc-feedback').addEventListener('opc-feedback:emit', (data) => console.log(data.detail.data));
